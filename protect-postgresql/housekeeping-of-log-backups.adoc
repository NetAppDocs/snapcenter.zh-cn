= SnapCenter如何管理数据备份的内务管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter在存储系统和文件系统级别管理数据备份的内务管理。

主存储或二级存储上的快照及其在PostgreSQL目录中对应的条目将根据保留设置被删除。
