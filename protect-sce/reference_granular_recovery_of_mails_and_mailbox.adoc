= 邮件和邮箱的粒度恢复
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
通过单邮箱恢复（ SMBR ）软件，您可以还原和恢复邮件或邮箱，而不是整个 Exchange 数据库。

仅为恢复单封邮件而还原完整数据库将占用大量时间和资源。SMBR 可通过创建 Snapshot 的克隆副本，然后使用 Microsoft API 在 SMBR 中挂载邮箱来帮助快速恢复邮件。有关如何使用 SMBR 的信息，请参见 https://["《 SMBR 管理指南》"^]。
