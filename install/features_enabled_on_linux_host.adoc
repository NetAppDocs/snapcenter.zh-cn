= 安装期间在Linux主机上启用的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter服务器会安装以下软件包、这些软件包可能适合进行故障排除和维护主机系统。

* Rabbitmq
* nginx
* Erlang
* 以8.0.5开头的.NET Framework、包括所有后续的.NET 8修补程序
* PAM-DEVEL
* PowerShell
