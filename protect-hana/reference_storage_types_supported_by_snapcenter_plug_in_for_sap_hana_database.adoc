= 适用于 SAP HANA 数据库的 SnapCenter 插件支持的存储类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter 在物理机和虚拟机（ VM ）上支持多种存储类型。在安装适用于 SAP HANA 数据库的 SnapCenter 插件之前，您必须验证是否支持您的存储类型。

|===
| 计算机 | Storage type 


 a| 
物理和虚拟服务器
 a| 
FC 连接的 LUN



 a| 
物理服务器
 a| 
iSCSI 连接的 LUN



 a| 
物理和虚拟服务器
 a| 
NFS-connected volumes

|===
