= 为 SAP HANA 资源定义还原和恢复策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须在还原和恢复数据库之前定义策略，以便成功执行还原和恢复操作。

.步骤
. 确定手动添加的 SAP HANA 资源支持的还原策略
. 确定自动发现的 SAP HANA 数据库支持的还原策略
. 确定要执行的恢复操作的类型。
