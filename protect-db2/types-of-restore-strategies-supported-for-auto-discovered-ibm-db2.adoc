= 自动发现的IBM DB2支持的恢复策略类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须先定义策略，然后才能使用 SnapCenter 成功执行还原操作。

完全资源恢复是自动发现的IBM DB2数据库支持的恢复策略。此操作将还原资源的所有卷、qtrees和LUN。
