= 在SnapCenter中创建Azure凭据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您应在SnapCenter中创建Azure凭据以访问Azure NetApp帐户。

在创建Azure凭据之前、请确保已在Azure中创建服务主体。要创建Azure凭据、需要提供与服务主体关联的租户ID、客户端ID和机密密钥。

.步骤
. 在左侧导航窗格中，单击 * 设置 * 。
. 在设置页面中，单击 * 凭据 * 。
. 单击 * 新建 * 。
. 在凭据页面中、指定创建凭据所需的以下信息。
+
|===
| 对于此字段 ... | 执行此操作 ... 


 a| 
凭据名称
 a| 
输入凭据的名称。



 a| 
身份验证模式
 a| 
从下拉列表中选择*Azure凭据*。



 a| 
租户ID
 a| 
输入租户ID。



 a| 
客户端 ID
 a| 
输入客户端ID。



 a| 
客户端机密密钥
 a| 
输入客户端密钥。

|===
. 单击 * 确定 * 。
