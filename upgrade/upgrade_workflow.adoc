= 升级工作流
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
每个 SnapCenter 版本都包含一个更新的 SnapCenter 服务器和插件软件包。插件软件包更新随 SnapCenter 安装程序一起分发。您可以将 SnapCenter 配置为检查可用更新。

工作流列出了升级SnapCenter服务器和插件包必须完成的任务。

image::../media/upgrade_workflow.gif[升级工作流]
