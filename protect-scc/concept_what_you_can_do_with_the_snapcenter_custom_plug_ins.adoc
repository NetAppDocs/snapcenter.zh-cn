= 您可以使用 SnapCenter 自定义插件执行什么操作
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 SnapCenter 自定义插件执行数据保护操作。

* 添加数据库，实例，文档或表空间等资源。
* 创建备份
* 从备份还原
* 克隆备份。
* 计划备份操作。
* 监控备份，还原和克隆操作。
* 查看备份，还原和克隆操作的报告。
