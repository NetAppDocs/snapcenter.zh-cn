= 如何以本机方式访问 SnapCenter REST API
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用支持 REST 客户端的任何编程语言直接访问 SnapCenter REST API 。常见语言选项包括 Python ， PowerShell 和 Java 。
