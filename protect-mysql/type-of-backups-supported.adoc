= 支持的备份类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
备份类型指定要创建的备份类型。SnapCenter支持对MySQL数据库使用基于Snapshot副本的备份类型。



== 基于 Snapshot 副本的备份

基于Snapshot副本的备份可利用NetApp Snapshot技术为MySQL数据库所在的卷创建联机只读副本。
