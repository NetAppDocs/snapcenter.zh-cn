= 使用命令行界面卸载适用于 Linux 或 AIX 的插件软件包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用命令行界面卸载适用于 Linux 的 SnapCenter 插件软件包或适用于 AIX 的 SnapCenter 插件软件包。

* 您需要的内容 *

* 确保已删除已计划的作业
* 确保所有正在运行的作业均已完成。


* 步骤 *

运行 _/custom_location/netapp/snapcenter/spl/installation/plugins/uninstall_ 以卸载。
